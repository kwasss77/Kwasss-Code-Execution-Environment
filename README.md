## Команды (пока мало)
1. math
2. print
4. exit

### Общее
Обратите внимание, что синтакс строгий и он не прощает ошибок.

### math
math - строго маленькими буквами. Это оператор, который решает примеры со знаками +, -, * (умножить), / (разделить). Внимание! Он поддерживает только однозначные целые числа. Он не засчитает -2, 2.5, 27 и другие. А только 0, 1, 2, 3... 9 . Пример синтаксиса: сторого "m 1+3" или "m 7-3". Команду писать без кавычек. Обратите внимание что есть только команда m и ставиться только один пробел перед примеров. С большой буквы не зарегает. Результат выводится в консольное окно Windows (если запускать командой оттуда же).

### print
print - команда которая выводит встроенные в язык переменные (ниже) или свой текст. Примеры команды (без ординарных кавычек): 'print "Привет, мир!! English text."'; 'print [math_title]'. Итоговый текст выводится в стандартое msg окно от Windows. Оно не поддерживает русский язык. изменено: можно писать текст в двойных кавычках (тогда внутри можно будет ставить ординарные), и наоборот. Также их вообще можно опустить (если вы не будете использовать кавычки в самом тексте).

### exit
exit - просто 4 буквы. Вы должны поставить эту команду в конец .kc2e файла во избежание ошибок и вылетов, и прочих багов.

## Как этим пользоваться???
Например, вы создали текстовый файл, переименовали его в test.kc2e, и написали простенькую программу:
print privet
m 1+4
exit
Понятно да, пишет текст, считает, выходит. Дальше, вам потребуется мой язык собственно говоря (файлом), ассемблер (программа-компилятор из языка "ассемблер" в машинный код) и линкер (сборщик в .exe). Для этих целей давайте возьмём Nasm и GoLink.
Скачивайте в 
Nasm - https://www.nasm.us/pub/nasm/releasebuilds/3.01/win64/ (для windows x64, версия 3.01) 
GoLink - https://www.godevtool.com/#64 (для x64, скачивайте GoAsm 32/64-bit assembler, ссылка на этот файл ниже)
файл: https://www.godevtool.com/Goasm.zip
