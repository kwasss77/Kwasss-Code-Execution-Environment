## Команды (пока мало)
1. math
2. print
3. open
4. exit
### Общее
Обратите внимание, что синтакс строгий и он не прощает ошибок.
### math
math - строго маленькими буквами. Это оператор, который решает примеры со знаками +, -, * (умножить), / (разделить). Внимание! Он поддерживает только однозначные целые числа. Он не засчитает -2, 2.5, 27 и другие. А только 0, 1, 2, 3... 9 . Пример синтаксиса: сторого "m 1+3" или "m 7-3". Команду писать без кавычек. Обратите внимание что есть только команда m и ставиться только один пробел перед примеров. С большой буквы не зарегает. Результат выводится в стандартое msg окно от Windows.

### print
print - команда которая выводит встроенные в язык переменные (ниже) или свой текст. Примеры команды (без ординарных кавычек): 'print "Привет, мир!! English text."'; 'print [math_title]'. Итоговый текст выводится в стандартое msg окно от Windows.
